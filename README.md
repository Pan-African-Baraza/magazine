Pan African Baraza magazine
===
Digital magazine for Pan-African Baraza

## Requirements
* ruby >= 2.0

## Setup

Clone the repository:

```
git clone https://github.com/Pan-African-Baraza/magazine.git && cd magazine
```

Install the gems:

```
bundle install
```

## Run

Start Jekyll:

```
jekyll serve -w
```

Point you browser to [http://localhost:4000/]()

Done :)
